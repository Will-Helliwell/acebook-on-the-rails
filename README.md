## To Use

### From Heroku

[Visit live deployment](https://acebook-on-the-rails.herokuapp.com/)

### From your local machine

First, clone this repository. Then:

```bash
> bundle install
> bin/rails db:create
> bin/rails db:migrate

> bin/rails server # Start the server at localhost:3000
```

To run tests:
```bash
> rspec
```
